Graph QL API

Overview

Created by Facebook to speed up its API by having a single endpoint that can take complex queries.

GraphQL is used to lead data from a server to a client using its syntax.

GraphQL lets the client specifiy what data it needs.

Makes it easier to aggregate data from multiple sources.

Uses a type system to describe data.

So what is GraphQL?

GraphQl is a query language. Wikipedia defines a query languange or data query languages (DQLs) as computer languages used to make queries in databases and information systems.

Using GraphQL a developer has a common interface between the client and the server for data fetching. What this means is that the client can ask for a variety of data from the GraphQL servier via queries. Using client-specified queries the response format is described in the query and defined by the client instead of the server.

The structure of the data is not hardcoded like traditional REST APIs, which makes retriving data from the server more efficient for the
